### ¿Porque GitFlow?

|Motivo|Desarrollo|
|------|-------------------|
|Aprender|Es una modalidad que seguramente sera más compleja y que usare seguido en las áreas que quiero trabajar|
|Legibilidad|Es más facil observar cambios al dividir el proyecto en varias ramas|
|*Gajes*|Es importante saber hacer las cosas bien desde el inicio. Es mas dificil quitarse las malas costumbres|
